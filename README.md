# AI & Career Navigation Datathon - Participant Packet

This repository contains the participant packet website for the AI & Career Navigation Datathon, hosted by the Gates Foundation and the Schultz Family Foundation.

## Project Overview

The datathon brings together leaders from across the career navigation field to build, test, and experiment with new approaches to career navigation. Using the American Opportunity Index (AOI) as a shared data foundation, participants prototype next-generation, public-use tools that help young adults and workers make more informed, equitable decisions about their futures.

## Files

- `index.html` - Main participant packet webpage (original version)
- `index_v2.html` - Updated version with improved design and tightened submission section

## Event Details

- **Dates:** January 28–29
- **Location:** Gates Foundation Headquarters, Seattle, WA
- **Focus:** AI-enabled career navigation tools using AOI data
